# Unity Package: Scene Switcher Window

![preview](https://media2.giphy.com/media/v1.Y2lkPTc5MGI3NjExdXVhMGU3dGhwanYya3Q1b205M3l4dnV3ZnR5NzVydjl0cTdmNW1oeiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/9AGCzk5n0r7SYVodkg/giphy.gif)

## Features
- Loads scenes directly or additively
- Loads scenes at runtime (for testing)
- Button to add the open scene to the build settings.

## Use Instructions
- Window > General > Scene Switcher
- Scenes need to be added to the build settings.
- There is an info foldout in case you need a quick refresher.

## Installation
### Package Manager
- Window > Package Manager
- \+ \> Install Package from git url
- `https://github.com/KWaldt/scene-switcher-window.git`

### Manual
Download the [Scene Switcher Window](https://github.com/KWaldt/scene-switcher-window/blob/main/Editor/SceneSwitcherWindow.cs) and put it into an **Editor** folder.
